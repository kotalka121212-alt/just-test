### Работа со строками

- [ ] `trim()` — убрать пробелы в начале и конце
- [ ] `trimLeft()` / `trimRight()` — убрать только слева или справа
- [ ] `split()` по разделителю
- [ ] `split()` по нескольким разделителям (пробел и запятая одновременно)
- [ ] `join()` — собрать вектор в строку с разделителем
- [ ] `replaceAll()` — заменить все вхождения подстроки
- [ ] `startsWith()` / `endsWith()` — проверка начала и конца строки
- [ ] `toUpperCase()` / `toLowerCase()`
- [ ] `capitalize()` — первая буква заглавная, остальные строчные
- [ ] `reverse()` — перевернуть строку
- [ ] `isPalindrome()` — проверка на палиндром
- [ ] `countWords()` — подсчёт слов
- [ ] `wrapText()` — перенос строк по заданной ширине
- [ ] `levenshteinDistance()` — расстояние между строками (похожесть)
- [ ] `soundex()` — фонетическое кодирование (похожесть по звучанию)
